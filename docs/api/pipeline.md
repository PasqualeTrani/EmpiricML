# empml.pipeline

| Object | Description |
| :--- | :--- |
| `Pipeline` | Custom pipeline for chaining transformers and an optional final estimator. |
| `relative_performance` | Compute the relative performance of a pipeline. |
| `train_pipeline` | Train the pipeline on training data. |
| `predict_with_pipeline` | Generate predictions using the pipeline. |
| `compute_score` | Compute metric score for predictions. |
| `eval_pipeline_single_fold` | Evaluate pipeline performance on a single fold. |
| `eval_pipeline_cv` | Evaluate pipeline performance using cross-validation. |
| `compare_results_stats` | Compute statistics comparing two sets of results. |
