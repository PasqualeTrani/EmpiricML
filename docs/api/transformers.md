# empml.transformers

| Object | Description |
| :--- | :--- |
| `Identity` | Pass-through transformer. |
| `AvgFeatures` | Compute mean across multiple features. |
| `MaxFeatures` | Compute max across multiple features. |
| `MinFeatures` | Compute min across multiple features. |
| `StdFeatures` | Compute standard deviation across multiple features. |
| `MedianFeatures` | Compute median across multiple features. |
| `KurtFeatures` | Compute kurtosis across multiple features. |
| `SkewFeatures` | Compute skewness across multiple features. |
| `ModuleFeatures` | Compute modulus of features. |
| `MeanTargetEncoder` | Target encoding using mean. |
| `StdTargetEncoder` | Target encoding using standard deviation. |
| `MaxTargetEncoder` | Target encoding using max. |
| `MinTargetEncoder` | Target encoding using min. |
| `MedianTargetEncoder` | Target encoding using median. |
| `KurtTargetEncoder` | Target encoding using kurtosis. |
| `SkewTargetEncoder` | Target encoding using skewness. |
| `OrdinalEncoder` | Encode categorical features as ordinal integers. |
| `DummyEncoder` | One-hot encode categorical features. |
| `StandardScaler` | Standardize features by removing the mean and scaling to unit variance. |
| `MinMaxScaler` | Transform features by scaling each feature to a given range. |
| `Log1pFeatures` | Apply log1p transformation. |
| `Expm1Features` | Apply expm1 transformation. |
| `PowerFeatures` | Apply power transformation. |
| `InverseFeatures` | Apply inverse transformation (1/x). |
| `SimpleImputer` | Impute missing values. |
| `FillNulls` | Fill null values with a specific strategy. |
| `GenerateLags` | Generate lag features for time series data. |
