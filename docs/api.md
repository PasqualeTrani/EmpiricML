# API Reference

This is the class and function reference of EmpiricML. Please select a module from the navigation menu or the table below to view its documentation.

| Module | Description |
| :--- | :--- |
| [`empml.base`](api/base.md) | Core abstract base classes for building other components. |
| [`empml.cv`](api/cv.md) | Cross-validation splitting strategies and utilities. |
| [`empml.data`](api/data.md) | Classes for downloading and loading data. |
| [`empml.errors`](api/errors.md) | Custom exceptions used throughout the framework. |
| [`empml.lab`](api/lab.md) | The main experimentation framework for model development. |
| [`empml.metrics`](api/metrics.md) | Performance metrics for regression and classification. |
| [`empml.pipeline`](api/pipeline.md) | Pipeline orchestration and evaluation tools. |
| [`empml.transformers`](api/transformers.md) | Feature engineering transformers compatible with Polars. |
| [`empml.wrappers`](api/wrappers.md) | Wrappers for integrating Scikit-learn and PyTorch models. |
